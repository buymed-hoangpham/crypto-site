# Crypto Website

Building website for Crypto!

# Technology

NextJS, Typescript, Module CSS, HTML5, CSS,...

# Using

<ol>
<li>
    Clone this repo to your computer
</li>
<li>
    Run `npm install` or `yarn` in your terminal
</li>
<li>
    Run `yarn start` or `npm start`
</li>
